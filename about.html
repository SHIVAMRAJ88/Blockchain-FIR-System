About FIRChain

FIRChain is a blockchain-based First Information Report (FIR) management system designed to ensure transparency, security, and trust in the law enforcement process.

üéØ Our Vision

To build a transparent and tamper-proof digital FIR system where citizens can trust the authenticity of law enforcement records.

üöÄ Our Mission

To eliminate FIR manipulation by leveraging blockchain technology and providing a secure, decentralized platform for FIR registration.

Why Blockchain?

  • Immutable FIR records (cannot be altered)
  • Decentralized verification
  • Enhanced data security
  • Full transparency & auditability

How FIRChain Works

  1. User logs in using OTP or MetaMask
  2. FIR details are submitted through secure form
  3. Data is encrypted and stored on blockchain
  4. FIR hash is generated for verification
  5. Authorized officials verify FIR authenticity

Technologies Used

HTML CSS JavaScript Blockchain Ethereum MetaMask Smart Contracts

Project Objective

The objective of FIRChain is to digitize FIR management, prevent data tampering, reduce delays, and ensure accountability using blockchain technology.
